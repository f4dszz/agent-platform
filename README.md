# Agent Platform

My multi-agent working chatroom/platform.

A multi-agent chat platform that orchestrates CLI-based AI agents (Claude Code, Codex CLI, etc.) through a web/mobile interface.

## Features (Planned)

- Multi-agent chat room with human participants
- CLI agent wrappers (Claude Code, Codex CLI)
- Real-time message streaming via WebSocket
- Session management with auto-rotation
- Permission and security controls
- Mobile-friendly web UI
